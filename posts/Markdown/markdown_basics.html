개요

마크다운이란?

마크다운(Markdown)은 마크업 언어(Markup Language)의 일종으로, 2004년에 존 그루버(John Gruber)가 만들었다. 일반 텍스트로 서식이 있는 문서를 작성하는 데에 사용되며, 문법이 간단하여 읽고 쓰기 쉽다.

Note

파일 확장자는 .md, .markdown이다.

마크다운 언어를 사용하는 이유

  1. Versatile: 마크다운은 웹사이트, 문서, 노트, 책, 프레젠테이션, 이메일 메시지 및 기술 문서 작성 등 다양한 용도로 사용할 수 있다.

  2. Portable: 마크다운 형식의 텍스트 파일은 거의 모든 응용 프로그램에서 열 수 있다.

  3. Platform independent: 모든 운영 체제에서 마크다운 형식 텍스트를 생성할 수 있다.

  4. Future-Proof: 애플리케이션이 작동을 멈춘다 하더라도 텍스트 편집 애플리케이션을 사용하여 마크다운 형식의 텍스트를 계속 읽을 수 있다.

  5. Support: Reddit, GitHub 등 많은 데스크탑과 웹 기반 애플리케이션에서 마크다운을 지원한다.

목록(Lists)

1. 순서 있는 목록

목록 숫자는 1로 시작해야 하며, 오름차순이 아니어도 된다.

Markdown 문법 출력 
1. First item
2. Second item
3. Third item
  1. First item
  2. Second item
  3. Third item
 
1. First item
1. Second item
1. Third item
  1. First item
  2. Second item
  3. Third item
 
1. First item
8. Second item
5. Third item
  1. First item
  2. Second item
  3. Third item
 
1. First item
2. Second item
    1. Indented item
    2. Indented item
3. Third item
  1. First item
  2. Second item
    1. Indent item
    2. Indent item
  3. Third item

2. 순서 없는 목록

-,*,+을 사용하여 순서 없는 목록을 생성할 수 있다.

Markdown 문법 출력 
- First item
- Second item
- Third item
  • First item
  • Second item
  • Third item
 
* First item
* Second item
* Third item
  • First item
  • Second item
  • Third item
 
+ First item
+ Second item
+ Third item
  • First item
  • Second item
  • Third item
 
- First item
- Second item
    - Indented item
    - Indented item
- Third item
  • First item
  • Second item
    • Indent item
    • Indent item
  • Third item

3. 목록에 요소(Element) 추가

목록에 다른 요소를 추가하려면 요소 앞에 들여쓰기 4칸을 삽입한다.

* First item
* Second item
    - Indented item

        > Blockquote
    
    - Indented item
* Third item
  • First item
  • Second item

    • Indented item

      Blockquote

    • Indented item

  • Third item

블록(Blocks)

1. 코드 블록(Code Blocks)

Fenced Code Blocks

코드 위와 아래를 ``` 또는 ~~~로 감싸 코드 블럭을 만들 수 있다.

```
a = 5
if a % 2 == 0:
  print("even")
else:
  print("odd")
```
a = 5
if a % 2 == 0:
  print("even")
else:
  print("odd")
구문 강조(Syntax Highlighting)

구문 강조를 위해서 첫 ``` 뒤에 코드 작성에 사용된 언어를 적는다.

```python
a = 5
if a % 2 == 0:
  print("even")
else:
  print("odd")
```
a = 5
if a % 2 == 0:
  print("even")
else:
  print("odd")
